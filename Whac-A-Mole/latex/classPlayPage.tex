The \mbox{\hyperlink{classPlayPage}{Play\+Page}} class for displaying the game interface. 

This class is responsible for setting up and displaying the game play interface. It includes player name input, start button, welcome label, countdown, and score display. \begin{DoxyAuthor}{Author}
