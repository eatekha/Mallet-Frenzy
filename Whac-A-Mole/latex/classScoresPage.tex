\doxysubsection{Detailed Description}
Class responsible for displaying high scores in the application. 

Class representing the high scores page in the GUI application.

This class sets up and displays a page showing high scores. It includes a QLabel for the title, a QList\+Widget for listing scores, and a QPush\+Button to return to the main menu. \begin{DoxyAuthor}{Author}
Yangxiuye Gu
\end{DoxyAuthor}
The \mbox{\hyperlink{classScoresPage}{Scores\+Page}} class is responsible for displaying high scores in a list format. It includes a return button to navigate back to the main menu.
